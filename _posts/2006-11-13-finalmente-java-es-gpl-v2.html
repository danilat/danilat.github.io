Lo leo en javahispano y en este momento está en portada de menéame .

Al parecer ahora se liberan Java SE y Java Me, mientras que Java EE (Glassfish) se liberará a principios de año.

Personalmente me ha extrañado mucho que la licencia sea GPL, pensaba que sería un tipo de licencia con más restricciones.

Es la licencia que más me gusta, que cualquiera haga las modificaciones o mejoras que crea necesarias, que puedan haber varias distribuciones... y la probable popularización de Java en la comunidad Linux.
