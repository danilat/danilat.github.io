Aparece en javaHispano la noticia de que se ha creado un grupo para preparar las certificaciones java de Sun.

La idea es preparar cualquiera de las certificaciones con el objetivo de , en navidad, conseguir la SCWCD(Sun Certified Web Content Developer ).

Yo ya me he apuntado en la lista, espero tener tiempo suficiente para poder plantearme el presentarme al menos al SCJP(Sun Certified Java Programmer).
