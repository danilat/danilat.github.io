Ya está disponible para descargar el Netbeans 5.5.

Además también se puede pedir el envío gratuito del cd.

Aquí podéis ver una introducción de esta versión del IDE de SUN.

Soporta estas tecnologías:

* Enterprise JavaBeans (EJB) 3.0
* JAX-WS 2.0
* Java Persistence (JSR-220)
* JavaServer Faces 1.2 (JSR-127)
* Java Servlet 2.5
* JavaServer Pages (JSP) 2.1
* Struts 1.2.9
* Java API for XML-based Remote Procedure Calls (JAX-RPC) 1.6
* Java APIs for XML Registries (JAXR) 1.0
* Java API for XML Processing (JAXP) 1.2
* JavaServer Pages Standard Tag Library 1.1

Habrá que probarlo.
